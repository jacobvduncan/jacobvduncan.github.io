Two Dimensional OpenGL Project - Winter 2024

Below is a project called plane attack that was developed continuously over the course of the winter for one of my course's final projects. It was made in C++ and OpenGL with no other external libraries. Below are some of the features this project has:

  • Movement transformations done with both parametric and physically based equations.
  • Multiple different enemy types with different looks, statistics, and behaviors.
  • Multiple different weapon types the player and AI can use.
  • A set of collectible items that give power-ups or affect player statistics, all reflected visually.
  • Custom-built UI to reflect time and other stats using hierarchical transformations of the player and different render passes.
  • Simple 2D collision detection using both circle-to-circle and ray-to-circle depending on the object.
  • Different particle and shader effects such as bloom, grayscale, and particles for bullet trails and explosions.
