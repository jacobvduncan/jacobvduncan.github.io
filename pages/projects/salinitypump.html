RTOS Salinity Pump - April 2025

For a fourth year operating systems class me and 2 other's loaded QNX, a proprietary blackberry RTOS onto a RPI to dynamically control the salinity levels in a solution/container (in our case just a jar).

Used two pumps, one to pump in freshwater, another to pump in saltwater.

  • Built a real-time salinity control system using QNX on Raspberry Pi, adjusting solution levels with two pumps based on live sensor readings.
  • Created a web interface with Node.js and Express.js to view and update salinity values, hosted on an Amazon EC2 server.
  • Set up reliable two-way communication between the Raspberry Pi and web server using periodic cURL GET and POST requests.
  • Configured QNX RTOS on the Raspberry Pi with working WiFi, networking, and GPIO control for pumps and sensors.
  • Connected and calibrated a TDS sensor with an ADC over I2C, integrating it into the system for accurate salinity measurement.
  • Handled full hardware setup, including pump control via relays, power supply wiring, and soldering stable connections to the Pi.
  • Managed real-time timing and logic in C to ensure responsive and accurate system behavior under a real-time operating system.
