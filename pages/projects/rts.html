Godot 3D RTS - March 2025

A 3D RTS made in Godot for a 4th year elective project, a timelapse of the gameloop can be found here.

  • Component-based architecture supporting dynamic unit behaviors using a sense-think-act model across farming, combat, and resource gathering systems.
  • Fully functional RTS building system with rotation support, grid-locked placement, and intelligent collision handling for multi-tile structures.
  • AI-driven enemy commander managing spawning strategies, unit states, and dynamic scaling for long gameplay sessions.
  • Advanced farming system with sowing, growth, harvesting, and unit cooperation, including fail-safe logic for edge cases like destroyed plots or resource contention.
  • Pathfinding implemented with real-time navigation mesh updates based on dynamic obstacles like walls, using NavigationAgent3D and NavigationRegion3D.
  • Isometric camera system with full player navigation controls (WASD, mouse drag, scroll zoom) and real-time feedback through responsive UI and sound design.
  • Complete game loop with victory/defeat conditions, freemode sandbox, full settings menu for audio/sensitivity, and contextual in-game feedback through visual and audio cues.
